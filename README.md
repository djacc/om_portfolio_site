# Portfolio Site

A simple portfolio website built with Next.js and deployed on Vercel.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the site.

### Build for Production

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Deployment

## Customization

Edit the files in the `app` directory to customize your portfolio:
- `app/page.tsx` - Main homepage content
- `app/layout.tsx` - Site layout and metadata
- `app/globals.css` - Global styles
